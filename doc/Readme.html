Get Started

 

After successful installation, use any batch file (in directory BCI2000\batch) to start the system with a particular configuration (e.g., RJB_RandomNumber.bat). Load then the corresponding parameter file (e.g., RJB_RandomNumber.prm) to configure the system before starting it. The batch files that contain RandomNumber will work on any machine; all others need data acquisition hardware (e.g., DTADC indicates the need for a A/D converter board from Data Translation).

 

Warning: Data Translation boards need to be named ‘BCI_IN’ (without quotation marks) before using the standard parameter files.  Use Settings->Control Panel->Open Layers Control Panel to rename the board.
